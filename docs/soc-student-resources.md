# 📚 10 Cybersecurity Resources with Student Discounts for SOC Analysts

A curated list of platforms, certifications, and tools that offer free tiers or student discounts — useful for anyone building SOC skills on a budget.

---

## 1. 🔐 TryHackMe
**What it is:** Beginner-friendly cybersecurity learning platform with guided labs and learning paths.
**Student discount:** Free tier available. Student discount via student email on Pro plan.
**Link:** [tryhackme.com](https://tryhackme.com)
**Best for:** SOC fundamentals, Splunk, Wireshark, defensive security paths.

---

## 2. 🔵 Blue Team Labs Online
**What it is:** Free platform focused entirely on defensive security and SOC skills.
**Student discount:** Completely free for most content.
**Link:** [blueteamlabs.online](https://blueteamlabs.online)
**Best for:** Log analysis, SIEM practice, malware analysis, incident response.

---

## 3. 🎓 Cybrary
**What it is:** Online cybersecurity training platform with structured career paths.
**Student discount:** Free tier + student pricing available.
**Link:** [cybrary.it](https://cybrary.it)
**Best for:** SOC Analyst career path, CompTIA prep, Splunk training.

---

## 4. 🧪 Hack The Box (HTB)
**What it is:** Hands-on offensive and defensive security labs.
**Student discount:** HTB Student plan available at reduced pricing.
**Link:** [hackthebox.com](https://hackthebox.com)
**Best for:** SOC labs, forensics challenges, Sherlocks (defensive challenges).

---

## 5. ☁️ Vultr Cloud
**What it is:** Cloud provider used in this challenge for running lab VMs.
**Student discount:** $300 free credit for new accounts (no student ID needed).
**Link:** [vultr.com](https://www.vultr.com/?ref=9632889-9J)
**Best for:** Hosting ELK Stack, Windows Server, and Ubuntu VMs for SOC labs.

---

## 6. 🔎 Elastic (ELK Stack)
**What it is:** The SIEM platform used throughout this challenge.
**Student discount:** Free tier on Elastic Cloud (14-day trial, then limited free tier).
**Link:** [elastic.co/cloud](https://www.elastic.co/cloud)
**Best for:** Learning Kibana, detection rules, dashboards, Fleet Server.

---

## 7. 📖 SANS Cyber Aces
**What it is:** Free foundational cybersecurity courses from SANS Institute.
**Student discount:** Completely free.
**Link:** [sans.org/cyberaces](https://www.sans.org/cyberaces/)
**Best for:** Networking, OS, and security foundations.

---

## 8. 🏆 KC7 Cyber
**What it is:** Free gamified cybersecurity investigation platform using KQL on Azure Data Explorer.
**Student discount:** Completely free.
**Link:** [kc7cyber.com](https://kc7cyber.com)
**Best for:** KQL query writing, threat hunting, SIEM investigation skills.

---

## 9. 🛡️ LetsDefend
**What it is:** SOC simulation platform with real-world alert investigation scenarios.
**Student discount:** Free tier available; paid plans at student pricing.
**Link:** [letsdefend.io](https://letsdefend.io)
**Best for:** Alert triage, SOC workflow, incident investigation practice.

---

## 10. 🔬 VirusTotal
**What it is:** Free online malware analysis and IOC lookup tool.
**Student discount:** Free to use (professional API has a free tier).
**Link:** [virustotal.com](https://virustotal.com)
**Best for:** Hash lookups, domain/IP reputation, malware sample analysis.
