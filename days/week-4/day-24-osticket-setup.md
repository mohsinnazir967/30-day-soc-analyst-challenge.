## 📋 Overview
Deploying and configuring osTicket — an open-source ticketing system — to simulate a SOC case management workflow.

## 🎯 Key Steps
- Deploy Ubuntu server for osTicket on Vultr
- Install LAMP stack (Apache, MySQL, PHP)
- Download and install osTicket
- Configure osTicket: departments, teams, help topics, SLA plans
- Create SOC-specific help topics: SSH Alert, RDP Alert, C2 Alert, Malware Detection
- Set up agent accounts for ticket assignment
- Access the osTicket staff panel
