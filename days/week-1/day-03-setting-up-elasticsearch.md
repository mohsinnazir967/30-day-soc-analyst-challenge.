**Week:** 1 | **Topic:** ELK Setup

## 📋 Overview
Installing and configuring Elasticsearch on Ubuntu, setting up network bindings, enabling security, and verifying the service is running correctly.

## 🎯 Key Steps
- Install Elasticsearch via apt package manager
- Configure `elasticsearch.yml` — network.host and cluster settings
- Enable xpack security (`xpack.security.enabled: true`)
- Start and enable the elasticsearch service
- Verify with `curl -X GET localhost:9200`
