# Day 07 — Setting Up Fleet Server and Installing Elastic Agent on Windows

**Week:** 1 | **Topic:** Agent Deployment
**Original file:** `Day 07 Setting Up Fleet Server and Installing Elastic Agent on Windows.md`

---

## 📋 Overview
Deploying Fleet Server on the ELK machine and enrolling the Windows Server as a managed endpoint using Elastic Agent.

## 🎯 Key Steps
- Set up Fleet Server in Kibana (Fleet → Settings)
- Generate the Fleet Server service token
- Install Fleet Server on the ELK machine
- Download and install Elastic Agent on Windows Server
- Enroll the Windows Agent into Fleet using the enrollment token
- Verify the agent appears as healthy in Kibana Fleet
