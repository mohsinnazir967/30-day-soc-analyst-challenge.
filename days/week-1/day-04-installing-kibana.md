## 📋 Overview
Installing Kibana, connecting it to Elasticsearch, and accessing the web UI for the first time.

## 🎯 Key Steps
- Install Kibana via apt package manager
- Edit `kibana.yml`: set `server.host` and `elasticsearch.hosts`
- Generate Kibana enrollment token from Elasticsearch
- Access Kibana at `http://server-ip:5601`
- Complete the initial setup wizard
