## 📋 Overview
Building a comprehensive Kibana dashboard to visualise RDP activity across the Windows Server — combining failed logins, successful logins, and source IP geolocation.

## 🎯 Key Steps
- Create visualisations: failed RDP login count over time, top attacking IPs, geo map
- Add successful RDP login tracking panel
- Combine all panels into a single RDP Activity dashboard
- Pin the dashboard to Kibana home for quick access
- Compare real internet-facing brute force traffic patterns
